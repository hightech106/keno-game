# Localization Files

Multi-language support files for the Keno game platform.

## Supported Languages

- English (en)
- French (fr)
- Spanish (es)
- Portuguese (pt)
- Arabic (ar) - RTL support
- Swahili (sw)
- Amharic (am)
- Tigrinya (ti)
- Oromo (om)

## File Structure

```
locales/
├── en.json
├── fr.json
├── es.json
├── pt.json
├── ar.json
├── sw.json
├── am.json
├── ti.json
└── om.json
```

Language files will be generated during Phase 4 implementation.
